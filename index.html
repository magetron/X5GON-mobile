Abstract

X5GON is an industrial leading Open Education Resource Provider. In this project, we worked with the UCL X5GON research team to deliver a mobile application that a Mobile Application that provides users with an authentic and mobile-friendly X5GON experience and learning materials catered for their needs.

With this project, the X5Learn system would be able to attract more users from the mobile platform, and enable them to learn anything anywhere.

FEATURES

X5GON was designed based on client requirements and user interviews. It offers all kinds of features and functions you would need on a learning platform.
